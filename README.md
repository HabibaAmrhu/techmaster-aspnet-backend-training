## How To Run

1. Clone the repository.
2. Navigate to the specific task folder (for example, `phase-02-web-api-basics/task-03-products-categories-api/`).
3. Run:

```bash
dotnet run
```

4. Access the interactive documentation:

```text
https://localhost:[PORT]/swagger
```
